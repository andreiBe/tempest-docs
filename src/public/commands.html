Welcome

This tool helps you easily set up and enjoy offline Paladins matches.

First Time Setup

  1. All standard maps are already included.
  2. No folder selection required. Launch the game and proceed.

Starting a Match

  1. Launch Paladins and wait at the login screen.
  2. Press F1 or F2 to open the console.
  3. Open the 🗺️ Offline Setup tab here.
  4. Select Map, Map Version & Game Mode; copy the generated command.
  5. Paste it into the in-game console and press Enter.

Playing

  1. After loading you'll be in spectator mode.
  2. Use the 🤖 Offline Tools tab to switch your champion.
  3. Spawn bots one-at-a-time using the spawn section.
Important: Spawn each bot individually. Copy & paste one command at a time.

Ending a Match

Use the Disconnect command under Quick Commands or press F1 at the end screen to return to login.
